12. Animation System

Use Framer Motion.

Motion should feel:

calm
premium
precise

Do not use:

Use this easing:

const premiumEase = [0.22, 1, 0.36, 1];

Core animations:

Section reveal
opacity 0 → 1
y 36 → 0
duration 0.75–0.9s
easing [0.22, 1, 0.36, 1]
Staggered children
stagger 0.08–0.12s
SVG stroke drawing

Use Framer Motion pathLength:

<motion.path
  initial={{ pathLength: 0, opacity: 0 }}
  whileInView={{ pathLength: 1, opacity: 1 }}
  transition={{ duration: 1.4, ease: "easeInOut" }}
/>
Floating decorative elements
y movement 4–8px
duration 6–12s
repeat infinity
reverse direction
very subtle
Stats
